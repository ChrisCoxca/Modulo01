# Portfolio Web & CV - Christopher

Este repositorio contiene mi portafolio profesional y Curriculum Vitae en formato web. El proyecto demuestra habilidades de desarrollo frontend, diseño de interfaces y estructura semántica de HTML5.

El sitio incluye navegación entre el perfil profesional (CV) y una *landing page* demostrativa para un emprendimiento personal.

## Ver el proyecto en vivo
Visita la página publicada en GitHub Pages aquí:
**(https://chriscoxca.github.io/Modulo01/)**

## Estructura del Proyecto
El sitio consta de dos páginas principales interconectadas:

1.  **`index.html` (CV Principal):**
    * Diseño estilo Harvard.
    * Contiene perfil profesional, educación (ESCOM), experiencia (Insaite) y formulario de contacto.
    * Uso de etiquetas semánticas: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`.

2.  **`cxc-store.html` (Proyecto Personal):**
    * Landing page para CXC Store.

## Tecnologías Utilizadas
* **HTML5:** Estructura semántica y formularios.
* **CSS3:** Diseño responsivo, alineación.
* **Git & GitHub:** Control de versiones.
